⚖PoE2 工具箱 · 功能介绍

流放之路2 集市增强的 Chrome 扩展，统一侧边栏，功能可逐项开关。国服 / 国际服数据相互隔离。

🔍搜索历史 侧边栏 tab · 国服+国际服

自动捕获你在集市里发起的每一次搜索，按时间倒序列表展示，最多保留 100 条、30 天。

  • 点击任意一条直接跳回当时的搜索结果页。
  • 鼠标悬停时右侧出现 ×，可单条删除；标题栏「清空」按钮可一键清空。
  • 展示联盟、搜索 ID 等关键参数标签，便于快速辨认。
触发时机：页面 URL 变化、提交表单、点击「搜索 / Search」按钮、首次进入页面时各记录一次。

国服与国际服的历史记录分别存储，互不干扰。

⭐收藏管理 侧边栏 tab · 国服+国际服

把常用的搜索条件收藏起来，分文件夹整理，支持拖拽归类与导入导出备份。

  • 「收藏当前搜索」把当前页面搜索条件存为一条收藏。
  • 「新建文件夹」自定义分类，收藏可在文件夹之间拖拽移动。
  • 「导入文件夹」支持从导出的 JSON 备份恢复，跨设备 / 跨账号迁移。
数据隔离：国服、国际服各自独立的收藏库，同一份导出文件可在两侧分别导入。

📋复制 PoB 结果行内联按钮 · 国服+国际服

在每条搜索结果行上注入一个「复制PoB」按钮，一键把该物品翻译成 Path of Building 可识别的文本格式并写入剪贴板。

  • 中文词缀会被翻译回英文（PoB 只认英文），翻译表首次使用时从远程下载约 5.7MB 原料并在本地建表缓存，14 天更新一次。
  • 物品数据由页面 fetch 劫持实时捕获，无需额外请求。
  • 词典缓存可手动导入 / 更新（见设置页）。
词典共用：翻译词典内容国服 / 国际服一致，共用一份缓存以节省空间。

📖查看词缀 侧边栏 tab · 国服+国际服

选中某物品类型后，展示该类型在 poe2db.tw 上的全部可出词缀（词缀池、来源分组）。

  • 类型识别用三种方式兜底：交易行 filters API、DOM 文字精确匹配、MAIN 世界 search 类别转发。
  • 国服读 poe2db 简体页（/cn/），国际服读繁体页（/tw/），缓存 24 小时。
  • 支持词缀池按来源（腐化、精华、渎灵等）筛选展示。
常见问题：某类型「暂不支持查看词缀」通常是 poe2db 没有独立词缀页，或该类型未在类别映射表中。

◆妄想症统计 侧边栏 tab · 国服+国际服 · 仅 POE2

输入一个 poe.ninja 构筑列表链接和账号数量（10–100），后台抓取前 N 个角色的 Megalomaniac（妄想症）天赋珠并统计词条出现次数。

  • 英文词条通过当前 PoB 词典反查中文展示。
  • 点击词条可查看天赋详情（stats 数值、购买 ID），并生成集市购买链接。
  • 购买链接 payload 的 count.value.min 固定为 2，只替换对应词条 filter。
  • 天赋树详情数据来自 poe2db，URL 可在面板里自定义，缓存复用。
  • 上次输入 / 上次结果 / 勾选状态会缓存，刷新侧边栏不丢。
限流说明：每两个角色之间固定等待 1200ms，遇到 poe.ninja 429 会停止后续请求并提示。账号越多耗时越久，属正常现象。

⚙通用说明

  • 侧边栏：右上角按钮依次为 刷新（重载当前 tab 数据）/ 清空（清当前 tab 数据）/ 设置（打开扩展设置页）。刷新和清空按当前 tab 是否支持自动显隐。
  • 功能开关：在设置页为每个功能分别配置国服 / 国际服是否启用。国服默认全开，国际服默认全关（未充分测试，需手动开启）。
  • 数据隔离：搜索历史、收藏、妄想症缓存、上次 tab 等所有用户数据按国服 / 国际服前缀独立存储，互不污染。
  • 加载方式：原生 JS，无构建步骤。改完代码后在 chrome://extensions 点扩展的「刷新」重载；content script 改动还需刷新目标页面。
